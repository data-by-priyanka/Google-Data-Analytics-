## Data-Inspired Decision Making

Data-inspired decision making means using data to guide decisions instead of relying only on intuition.

The goal of a data analyst is to:
- Use accurate and relevant data
- Interpret it correctly
- Provide clear recommendations

Having complete and correct data is important. However, even good data can lead to wrong decisions if it is interpreted poorly.

There is an important difference between:
- **Incomplete data** – Missing important information (risky)
- **Small but accurate data** – Limited data from a controlled test (can still be useful)

Small, reliable data can support good decisions. Incomplete data can lead to costly mistakes.

## Algorithm

An algorithm is a step-by-step set of rules used to complete a task or solve a problem.

In data analytics, algorithms are often used to process data and identify patterns.

## Types of Data

### Quantitative Data

Quantitative data includes numbers and measurable facts.

It answers questions like:
- What?
- How many?
- How often?

Examples:
- Sales revenue
- Number of customers
- Website visits

Quantitative data helps us measure and compare results.

---

### Qualitative Data

Qualitative data describes qualities or characteristics that cannot be measured with numbers.

It answers questions like:
- Why?
- How do people feel?
- What is the experience like?

Examples:
- Customer feedback
- Reviews
- Opinions

Quantitative data shows the “what.”  
Qualitative data explains the “why.”

Data analysts usually use both together for deeper insights.

## Data Presentation Tools

### Reports

A report is a static summary of data shared regularly with stakeholders.

#### Advantages:
- Good for historical data
- Easy to design
- Useful for periodic updates
- Based on cleaned and finalized data

#### Disadvantages:
- Not interactive
- Require regular updates
- Do not show live data

#### Pivot Table

A pivot table is a tool used to summarize and organize large datasets.

It allows you to:
- Group data
- Count values
- Calculate totals or averages
- Rearrange rows and columns

---

### Dashboards

A dashboard is an interactive tool that displays live or frequently updated data.

It allows stakeholders to:
- Monitor performance
- Explore data with filters
- View multiple metrics in one place

#### Advantages:
- Dynamic and interactive
- Low maintenance once built
- Useful for continuous monitoring

#### Disadvantages:
- Time-consuming to design
- Can be overwhelming if poorly structured
- May require maintenance if data sources change

---

### Types of Dashboards

- **Strategic** – Focus on long-term goals and high-level metrics.
- **Operational** – Track short-term performance and daily activities.
- **Analytical** – Used for deeper analysis, predictions, and detailed exploration.

Dashboards help companies:
- Track performance
- Set goals
- Monitor key performance indicators (KPIs)
- Identify inefficiencies

---

## Metrics

A metric is a measurable value used to track performance.

Raw data becomes useful when organized into metrics.

Examples:
- Revenue
- Conversion rate
- Customer retention rate

### Metric Goal

A metric goal is a measurable target set by a company.

For example:
- Increase revenue by 10%
- Improve customer satisfaction score

Metrics help businesses measure progress toward goals.

---

## Mathematical Thinking

Mathematical thinking means breaking a problem into logical steps.

It helps you:
- Identify relationships in data
- Choose the right tools
- Analyze patterns effectively

This approach improves problem-solving accuracy.

---

## Small Data vs Big Data

### Small Data

Small data:
- Focuses on specific metrics
- Covers a short time period
- Is used for small-scale decisions

Example:
Tracking daily exercise time to burn 400 calories.

---

### Big Data

Big data:
- Involves large and complex datasets
- Covers longer time periods
- Is used for large-scale business decisions

Big data helps organizations:
- Identify customer behavior patterns
- Improve efficiency
- Make strategic decisions
- Monitor brand performance

---

## Challenges of Big Data

- Data overload
- Difficulty finding relevant information
- Accessibility issues
- Risk of algorithmic bias
- Technology limitations

---

## Benefits of Big Data

- Better decision-making
- Improved efficiency
- Stronger understanding of market trends
- Better customer insights
- Competitive advantage

---

## The 4 Vs of Big Data

- **Volume** – The amount of data.
- **Variety** – Different types of data.
- **Velocity** – Speed of data processing.
- **Veracity** – Accuracy and reliability of data.

All four are important when working with large datasets.

---

## Key Takeaway

Effective decision-making requires:
- The right data
- The right metrics
- Clear interpretation
- Logical thinking

Data alone does not create value. Insight does.
